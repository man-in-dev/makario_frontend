## Animation Effects

### Hover Effects
1. **Cards:** Shadow increases, background lightens
2. **Buttons:** Color changes, slight upward movement (-translate-y-1)
3. **Images:** Scale up (110%) on hover
4. **Icons:** Scale up (110%) on hover
5. **Text:** Color transitions smoothly

### Tab Switching
- Smooth color transitions
- Active button: Filled background
- Inactive buttons: Border only
- Content fade-in (via opacity)

### Transitions Used
- `duration-300` - Default smooth transitions
- `duration-500` - Slower animations for larger changes
- All with `ease-in-out` for natural movement
